# portfolio-frontend-cpp

Landing Page – Portfólio Pessoal Interativo
📌 Descrição

Landing page desenvolvida em HTML, CSS e JavaScript com o objetivo de apresentar meu perfil profissional e reunir diversos projetos web interativos em um único ambiente.
O portfólio funciona como um hub de aplicações front-end, contendo links para projetos práticos desenvolvidos durante minha formação em Sistemas de Informação.
O projeto demonstra organização estrutural, identidade visual e integração entre múltiplas aplicações web.

🚀 Funcionalidades da Landing Page
Seção "Sobre Mim" com apresentação profissional;
Layout estilizado e organizado em seções;
Menu de navegação com âncoras internas;
Seção de habilidades com barras de progresso;
Formulário de contato;
Estrutura preparada para responsividade;
Cards interativos que direcionam para projetos internos;
Projetos Integrados ao Portfólio;

A landing page reúne diversas aplicações desenvolvidas por mim, como:

Calculadora online;
Jogo da memória;
Lista de tarefas (To-Do List);
Sistema estilo Kanban;
Pedra, Papel e Tesoura;
Jogo de cartas;
Simulação de sistema bancário;
Jogo de número oculto;

Esses projetos envolvem lógica de programação, manipulação do DOM e interatividade com JavaScript.

🛠 Tecnologias Utilizadas

HTML5;
CSS3;
JavaScript;
Estruturação semântica de páginas;
Manipulação do DOM;
Organização modular de arquivos

🧠 Conceitos Aplicados

Estruturação de layout responsivo;
Navegação por âncoras;
Componentização visual (cards e seções);
Manipulação dinâmica de elementos;
Organização de múltiplos projetos dentro de uma única aplicação principal

🎯 Objetivo do Projeto

Criar um portfólio digital completo que centralize projetos práticos e demonstre evolução técnica em desenvolvimento web front-end.
